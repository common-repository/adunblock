== Description ==

AdUnblock affichage un message aux internautes ayant un bloqueur de publicité pour les inciter à désactiver celui-ci.

== Installation ==

1. Upload the folder `adunblock` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Register to adunblock.com
4. Put your site ID
